# AQI Analysis Dashboard (Power BI)

## 📊 Overview
This project analyzes Air Quality Index (AQI) across India.

## 🎯 KPIs
- Average AQI
- Most Polluted City
- AQI Category

## 🛠 Tools
- Power BI
- Excel
  
## 📈 Insights
- Pollution is highest in winter
- Major cities dominate AQI levels
